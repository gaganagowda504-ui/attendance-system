# Attendance Monitoring System Using Face Detection

A Machine Learning web application that detects faces through webcam and automatically marks attendance with date and time.

---

## ⚙ Tech Stack

- Python  
- Flask (Backend API)  
- OpenCV  
- React.js (Frontend)  
- HTML  
- CSS  
- JavaScript  

---

## ✧ Features

- Webcam face detection  
- Automatic attendance marking  
- Attendance table view  
- Date and time tracking  
- CSV attendance download  
- Local storage data saving  
- Responsive frontend UI
